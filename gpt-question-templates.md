# GPTに質問するためのIPO分析テンプレート集

## 基本的なIPO分析質問テンプレート

| Input | Process | Output |
|-------|---------|--------|
| 「[開発したいプログラム]の基本的な処理の流れを日本語で説明してください。特に[重視したい点]に注目して、[目的]のために必要なステップを教えてください。」 | GPTが処理フローを分析し、日本語での手順を考える | ・プログラムの処理フローの説明<br>・重要ステップの詳細<br>・考慮すべき点 |
| 「[プログラム名]のインプットとアウトプットを詳細に定義したいです。インプットとして必要なデータ、パラメータ、設定と、アウトプットとして生成されるデータ、ファイル、通知について教えてください。」 | GPTが入出力仕様を分析する | ・入力データの詳細定義<br>・出力データの詳細定義<br>・インターフェース仕様 |
| 「[プログラム]の主要な機能を細分化したいです。大きな処理を小さな機能単位に分割し、各機能の役割と相互関係を整理してください。特に[特定の機能]については詳しく教えてください。」 | GPTが機能を細分化し、関係性を分析する | ・機能の一覧と説明<br>・機能間の関係図<br>・特定機能の詳細分析 |
| 「[機能名]のIPO分析を行いたいです。この機能に必要な入力（Input）、処理内容（Process）のステップ、そして期待される出力（Output）を詳細に教えてください。エラーケースについても考慮してください。」 | GPTがIPO分析を実施する | ・入力の詳細<br>・処理ステップの詳細<br>・出力の詳細<br>・エラーケースの分析 |
| 「[プログラム]を実装する際に注意すべき重要なポイントを教えてください。特に[エラー処理/セキュリティ/パフォーマンス]について詳しく知りたいです。また、疑似コードで例示してもらえますか？」 | GPTが重要ポイントを分析し、疑似コードを生成する | ・注意点のリスト<br>・特定分野の詳細解説<br>・疑似コード例 |
